# 🎯 Life Quest

> **برنامه‌ریزی امروز، ساختن آینده.**

**Life Quest** یک اپلیکیشن مدیریت اهداف، برنامه‌ریزی روزانه و افزایش بهره‌وری است که به شما کمک می‌کند بدانید هر روز باید چه کارهایی انجام دهید، روی چه هدفی تمرکز کنید و پیشرفت خود را به‌صورت منظم دنبال کنید.

این برنامه با ترکیب مدیریت وظایف، هدف‌گذاری، عادت‌سازی و گزارش‌های تحلیلی، به شما کمک می‌کند هر روز یک قدم به اهداف بزرگ‌ترتان نزدیک شوید.

---

# ✨ امکانات

- ✅ مدیریت کارهای روزانه (To-Do)
- 🎯 تعیین اهداف کوتاه‌مدت و بلندمدت
- 📅 برنامه‌ریزی روزانه، هفتگی و ماهانه
- 📈 نمایش میزان پیشرفت اهداف
- 🔥 سیستم حفظ استریک (Streak)
- ⏰ یادآوری هوشمند
- 📊 گزارش و آمار عملکرد
- 📝 ثبت یادداشت‌های روزانه
- 🌙 حالت تاریک و روشن
- ☁️ همگام‌سازی اطلاعات
- 🔒 امنیت اطلاعات کاربران

# 🎯 اهداف (Goals)

در Life Quest می‌توانید اهداف مختلفی تعریف کنید، مانند:

- یادگیری زبان
- ورزش
- مطالعه کتاب
- برنامه‌نویسی
- مهاجرت
- پس‌انداز
- راه‌اندازی کسب‌وکار
- کاهش یا افزایش وزن
- یا هر هدف شخصی دیگر

برای هر هدف می‌توانید:

- تاریخ شروع و پایان تعیین کنید.
- میزان پیشرفت را ثبت کنید.
- وظایف مرتبط ایجاد کنید.
- یادآور تنظیم کنید.

# ✅ مدیریت وظایف

هر روز می‌توانید:

- وظایف امروز را مشخص کنید.
- اولویت هر کار را تعیین کنید.
- زمان انجام آن را ثبت کنید.
- پس از انجام، آن را تکمیل کنید.

نمونه:

- ☐ مطالعه آلمانی
- ☑ باشگاه
- ☐ یادگیری Git
- ☑ نوشیدن ۲ لیتر آب
- ☐ مطالعه کتاب
